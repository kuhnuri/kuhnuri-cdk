# AWS Batch based DITA-OT

## Prerequisites

To build and deploy, the following tools are required:

-  AWS CLI
-  AWS Account and User, with _Programmatic access_ type and `AdministratorAccess` policy
-  Node.js

Bootstrap CDK with:

```bash
npm run cdk bootstrap
```

## Building

```bash
npm run build
```

## Deployment

```bash
npm run cdk deploy
```
